# Kongre Yönetimi Sistemi

Bu paket, yüklenen çalışan yönetim sistemi şablonunun aynı tek parça HTML/CSS/JS düzeni korunarak kongre ve organizasyon yönetimi için yeniden yapılandırılmış sürümüdür.

## Giriş

- Kullanıcı adı: `admin`
- Şifre: `Sdkongre2026!`

## Kurulum

1. Dosyaları GitHub repository ana dizinine yükleyin.
2. Streamlit Cloud'da `app.py` dosyasını seçerek deploy edin.
3. Uygulama açılınca yukarıdaki kullanıcı bilgileriyle giriş yapın.

## Notlar

- Bu sürüm, önceki çalışan uygulamadaki gibi tarayıcı localStorage mantığıyla çalışır.
- Gerçek Firebase/Supabase bağlantısı bu pakette zorunlu değildir.
- Dashboard verileri sabit/demo değildir; kayıt ekranlarından oluşturulan gerçek kayıtlardan hesaplanır.
- Kaldırılması istenen modüller eklenmemiştir.
